📋 REQUIRED SETUP (One-time, all platforms)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Copy .env template:
       cp .env.example .env

  2. Edit .env and add your Copilot API key:
       COPILOT_API_KEY=sk-ant-xxxxx
       (Get one from: https://console.anthropic.com/)

  3. Save and run the launch script

🔧 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  "Python/Node not found?" → Install from python.org / nodejs.org
  "Permission denied?"     → chmod +x run.sh (macOS/Linux)
  "Port already in use?"   → Close other applications
  "Dashboard blank?"       → Wait 30 seconds, refresh browser

  Full troubleshooting: See LAUNCH_GUIDE.md
